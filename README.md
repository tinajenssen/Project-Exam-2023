A modern front end social media application for students at Noroff.
The websites goal is to unify the school community, by providing a
platform to share and discover inspiring content. The platform is inviting and fun, while also being professional and credible. The goal is that it is a place where Noroff students can come together and make meaningful connections, while being inspired by each otherâ€™s work.

CSS Frameworks: Bootstrap
Hosting Services: Netlify
